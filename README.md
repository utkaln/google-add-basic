# google-add-basic


## How to Run
1. Create Virtual Environment
    ```sh
    python3 -m venv .venv
    ```
2. Activate Virtual Environment
    ```sh
    source .venv/bin/activate
    ```
2. Install libraries
    ```sh
    pip install -r requirements.txt
    ```
3. (Optional) Deactive Virtual Environment
    ```sh
    deactivate
    ```
